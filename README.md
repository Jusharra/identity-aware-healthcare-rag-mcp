# GRC DevSecOps + MLSecOps Platform Template

This is the baseline template used for all future labs and engineering projects.

### Included:
- `platform/governance` — control catalog, risk register, policies-as-code  
- `platform/devsecops` — CI/CD templates, IaC modules, Python tooling  
- `platform/mlsecops` — ML security templates, model card, drift configs  

### Use:
Click **“Use this template”** in GitHub to generate a new governed lab.
